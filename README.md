# Travel Planner Android Application
## 1. Introduction

Travel Planner is an Android application developed to help users explore popular tourist destinations in India.
The app provides information about different states and their famous places, including an overview, things to do, best time to visit, hotels, and location details.

This project is created for academic and practical learning purposes, focusing on Android fundamentals.

## 2. Objectives

- To design a user-friendly travel information app

- To understand RecyclerView and Adapter implementation

- To implement Activity navigation using Intents

- To display structured data using custom layouts

- To integrate Google Maps for location navigation

## 3. Features

List of tourist destinations:

* Uttarakhand

* Goa

* Delhi

* Manali

* Jaipur

> State-wise places listing

> Detailed place information screen

> Tabs for: Overview | Things To Do

## 4. Application Flow

➡ User opens the app

➡ Main screen shows list of states

➡ Selecting a state opens its places list

➡ Selecting a place opens detailed information

➡ User can switch between Overview and Things To Do

➡ User can open the place location in Google Maps
